# Spam--Detection-System

This project detects whether an SMS message is spam or not using Machine Learning.

## Features
- SMS spam classification
- TF-IDF vectorization
- Naive Bayes classifier
- Web interface using Flask

## Technologies Used
- Python
- Scikit-learn
- Flask
- HTML/CSS

## Dataset
SMS Spam Collection Dataset.

## How to Run

1. Clone the repository
git clone https://github.com/darbhamullasainikhitasri23cse-stack/Spam--Detection-System.git

2. Install dependencies
pip install -r requirements.txt

3. Run the app
python app.py
